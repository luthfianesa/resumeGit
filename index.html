  • kemudian, add, commit, push master
  • di master, merge feature-b => master
  • di master 'resolve conflict di master'

Bagaimana Menyelesaikan Konflik Penggabungan (merge) di Git?

  • Cara termudah untuk menyelesaikan file yang berkonflik adalah dengan membukanya dan membuat perubahan yang diperlukan
  • Setelah mengedit file, kita dapat menggunakan perintah git add untuk menampilkan konten gabungan baru
  • Langkah terakhir adalah membuat komit baru dengan bantuan perintah git commit
  • Git akan membuat komit gabungan baru untuk menyelesaikan penggabungan
